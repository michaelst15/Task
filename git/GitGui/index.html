Git Intro

Pengertian Git

GIT adalah salah satu version control system yang diciptakan oleh Linus Torvalds dan GIT mencatat setiap perubahan file di code atau sourcode kita.

Tools Git
  1. GIT
  2. Akun Github
  3. Sourcetree
  4. Text Editor

Git Fundamental

Perintah perintah dasar yang sering dilakukan untuk mengoperasikan GIT
  1. Git Init

    2. Digunakan untuk menginisialisasi repository

  2. Git Config

    3. untuk Konfigurasi email dan username

  3. Git Status

    4. Digunakan untuk mengetahui status file

  4. Git add

    5. Digunakan untuk memasukkan file ke staging area

  5. Git commit

    6. Digunakan untuk mencatat setiap perubahan

  6. Git log

    7. Digunakan untuk menampilkan setiap commit atau perubahan

  7. Git clone

    8. Digunakan untuk mengcopy repo dari server

  8. Git remote

    9. Semacam api untuk berinteraksi dari client dengan server

  9. Git push

    10. untuk mengirim commitan ke server

  10. Git pull

    11. untuk menggambil commitan dari server
